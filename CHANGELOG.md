## [v2.0.0] - 2026-06-04

### Breaking
- Изменён способ просмотра документов: теперь `.docx` обрабатываются через серверный рендер, а не через простое предупреждение.
- Изменена логика отображения документов в просмотрщике: HTML-рендер стал ближе к Word-структуре.

### Added
- Добавлен рендер `.docx` с таблицами, изображениями и абзацами.
- Добавлена системная тема.
- Добавлены режимы работы: чтение и редактирование.
- Добавлен Word-подобный просмотр документов в просмотрщике.
- Добавлен режим сохранения настроек в `localStorage`.
- Добавлено отображение встроенных изображений в `.docx`.
- Добавлена поддержка таблиц в `.docx` и Excel-файлах.
- Добавлена серверная конвертация `.doc` через LibreOffice headless.
- Добавлен fallback на извлечение текста из `.doc`, если конвертация не удалась.

### Removed
- Удалён упрощённый просмотр без попытки сохранить структуру документа.

### Fixed
- Исправлено отображение таблиц в `.docx`.
- Исправлено отображение изображений внутри `.docx`.
- Исправлены пустые абзацы и пробелы в рендере документов.
- Исправлено отображение файлов в тёмной и светлой теме.
- Исправлена логика переключения режима редактирования.
- Исправлена навигация назад/домой в файловом браузере.
- Исправлены потенциальные ошибки при выводе HTML-контента в просмотрщике.
- Исправлена обработка путей, чтобы снизить риск выхода за пределы `BASE_DIR`.

### Security
- Усилена проверка путей через `path.resolve()` и контроль префикса базовой директории.
- Ограничен доступ к файлам вне корневой папки УМК.

### Performance
- Оптимизировано получение статистики по каталогу.
- Улучшен рендер документов за счёт серверной конвертации только по запросу.
- Добавлен более быстрый просмотр DOCX без необходимости ручной конвертации перед открытием.

### Documentation
- Добавлена документация

### Tests
- Проверка открытия `.docx` с таблицами.
- Проверка открытия `.docx` с изображениями.
- Проверка переключения тем.
- Проверка режима чтения.
- Проверка режима редактирования.
- Проверка скачивания файлов.
- Проверка открытия папок и возврата назад.
