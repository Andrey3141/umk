# FAQ.md

## [ERROR-9] - Не открывается файл в приложении
### Присутствует в версиях: [v2.3.0]
### Симптомы: При клике на файл с режимом "Открыть в приложении" ничего не происходит или появляется ошибка
### Причина: На компьютере не установлена программа для открытия данного типа файлов
### Решение:
1. Убедиться, что на компьютере установлена программа для открытия файла
2. Для DOCX/DOC — установить Microsoft Word или LibreOffice
3. Для XLSX/XLS — установить Microsoft Excel или LibreOffice Calc
4. Для PDF — установить Adobe Acrobat Reader или браузер с поддержкой PDF

## [ERROR-8] - Не отображается тёмная тема в просмотрщике
### Присутствует в версиях: [v2.1.0] - текущий
### Симптомы: При переключении на тёмную тему просмотрщик остаётся светлым
### Причина: Старая версия CSS или кеш браузера
### Решение:
1. Обновить страницу (Ctrl+F5)
2. Очистить кеш браузера
3. Переключить тему в настройках заново

---

## [ERROR-7] - Не сохраняется путь к УМК
### Присутствует в версиях: [v2.1.0] - текущий
### Симптомы: При сохранении пути в настройках появляется ошибка
### Причина: Нет прав на запись в `config.json` или папка не существует
### Решение:
1. Проверить права на запись в папке проекта
2. Убедиться, что указанная папка существует
3. Проверить, что у пользователя есть доступ к папке

---

## [ERROR-6] - Не отображаются таблицы в Excel
### Присутствует в версиях: [v2.1.0] - текущий
### Симптомы: Excel-файл открывается, но таблицы пустые или без форматирования
### Причина: Сложное форматирование или повреждённый файл
### Решение:
1. Убедиться, что файл имеет расширение `.xlsx` или `.xls`
2. Попробовать открыть файл в Excel и сохранить заново
3. Скачать файл для просмотра в Excel

---

## [ERROR-5] - Не проигрывается видео
### Присутствует в версиях: [v2.1.0] - текущий
### Симптомы: Видео не загружается или показывает чёрный экран
### Причина: Браузер не поддерживает формат или файл повреждён
### Решение:
1. Использовать современные форматы: MP4, WEBM
2. Проверить, что файл не повреждён
3. Скачать видео через кнопку "Скачать"

---

## [ERROR-4] - Не открывается PDF
### Присутствует в версиях: [v2.1.0] - текущий
### Симптомы: При открытии PDF пользователь видит предупреждение или пустой экран
### Причина: PDF защищён паролем, повреждён, или не установлен `pdf-parse`
### Решение:
1. Проверить, что установлен `pdf-parse`: `npm install pdf-parse`
2. Убедиться, что PDF не защищён паролем
3. Если файл не читается — скачать через кнопку "Скачать"

---

## [ERROR-3] - Не отображаются таблицы и картинки в документе
### Присутствует в версиях: [v1.0.0] - текущий
### Симптомы: Пользователь открывает `.docx`, но видит только часть текста
### Причина: Документ имеет сложную структуру
### Решение: 
1. Обновить проект до последней версии
2. Проверить, что файл имеет расширение `.docx`
3. Если документ сложный — часть форматирования может не совпадать

---

## [ERROR-2] - `.doc` не открывается как документ
### Присутствует в версиях: [v1.0.0] - текущий
### Симптомы: При открытии `.doc` пользователь видит предупреждение
### Причина: Старый формат Word
### Решение:
1. Убедиться, что на сервере установлен LibreOffice
2. Установить: `sudo apt install libreoffice` (Linux)
3. Или скачать с https://www.libreoffice.org/ (Windows)
4. Если файл не конвертируется — сохранить его как `.docx`

---

## [ERROR-1] - Не запускается приложение
### Присутствует в версиях: [v1.0.0] - текущий
### Симптомы: При запуске сервер не стартует
### Причина: Не установлены зависимости или не найдена папка с УМК
### Решение:
1. Проверить, что установлен Node.js 18+
2. Выполнить `npm install`
3. Убедиться, что существует путь к папке УМК
4. Запустить сервер: `npm start`
