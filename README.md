## 📖 О проекте

**УМК МГТК** — это веб-платформа и **полноценное Windows-приложение** для просмотра и навигации по учебно-методическим материалам.  
Проект позволяет открывать папки, просматривать документы, скачивать файлы, открывать их в системных приложениях (Word, Excel, PDF-ридер) и работать с материалами в удобном интерфейсе.

### Что это даёт
- 📁 удобный доступ к материалам из браузера или приложения
- 📄 просмотр всех основных форматов: `docx`, `doc`, `xlsx`, `pdf`, изображения, видео
- 💻 **открытие документов в Word/Excel/PDF-ридере** (режим "Открыть в приложении")
- 🎨 поддержка светлой, тёмной и системной темы
- ✏️ режим чтения и режим редактирования текстовых файлов
- 📦 **готовое Windows-приложение** (установщик .exe)
- 🔒 **не требует прав администратора**

## 📥 Установка и запуск

### Вариант 1: Готовое Windows-приложение (рекомендуется)
1. Скачать `УМК Мастер Setup 1.0.0.exe`
2. Запустить установку (не требует прав администратора)
3. Нажать на ярлык на рабочем столе
4. Приложение автоматически откроет сайт в браузере

### Вариант 2: Запуск из исходников (Linux/Windows)
```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm start

# Сборка Windows-приложения (32-bit)
npm run dist:win32

# Сборка Windows-приложения (64-bit)
npm run dist:win64
```

После запуска откройте: `http://localhost:3000`

## 🎮 Использование

### Основные действия
1. Откройте корневую папку УМК (настраивается в интерфейсе)
2. Перейдите в нужный раздел
3. Откройте документ кликом по файлу
4. Используйте настройки для смены темы или режима
5. При необходимости скачайте файл или откройте в приложении

### Настройки
- **📂 Путь к УМК:** укажите папку с материалами
- **🎨 Тема:** светлая, тёмная, системная
- **👆 Действие при клике:** просмотр на сайте, скачивание, открыть в приложении
- **✏️ Режим:** чтение или редактирование

### Что умеет просмотрщик
- ✅ Word-документы (DOCX, DOC) с таблицами и картинками
- ✅ Excel-таблицы с форматированием
- ✅ PDF с извлечением текста и разбивкой на страницы
- ✅ Видео во встроенном плеере
- ✅ Изображения с масштабированием
- ✅ Текстовые файлы с подсветкой
- ✅ Тёмная тема для всех типов файлов
- ✅ **Кнопка "Открыть в приложении"** в просмотрщике

## 🛠 Технологии

```text

Express         — HTTP API и маршрутизация
Electron        — Windows-приложение-обёртка

```

## 🔧 Решение проблем

<details>
<summary>❌ Не запускается сервер</summary>

- Проверьте, что установлен Node.js 18+
- Выполните `npm install`
- Убедитесь, что порт 3000 свободен
</details>

<details>
<summary>❌ Не открываются DOC-файлы</summary>

- Проверьте наличие LibreOffice или `soffice`
- Убедитесь, что файл не повреждён
- Попробуйте сохранить файл как `.docx`
</details>

<details>
<summary>❌ Не отображаются PDF</summary>

- Проверьте, что установлен `pdf-parse`: `npm install pdf-parse`
- Убедитесь, что PDF не защищён паролем
- Если файл не читается — скачайте его через кнопку
</details>

<details>
<summary>❌ Не работает видео</summary>

- Проверьте, что браузер поддерживает формат
- Используйте современные форматы: MP4, WEBM
- Убедитесь, что файл не повреждён
</details>

<details>
<summary>❌ Не сохраняется тема или путь</summary>

- Настройки сохраняются в `localStorage` и `config.json`
- Проверьте права на запись в папке проекта
</details>

<details>
<summary>❌ Не открывается файл в приложении</summary>

- Убедитесь, что на компьютере установлена программа для открытия файла
- Например, для DOCX нужен Word или LibreOffice
- Для PDF — любой PDF-ридер
</details>
